# Soil Moisture Checker using Op-Amps

A microcontroller-free, purely analog soil moisture monitoring system built with an LM358 window comparator (evaluates voltage against set limits). It uses a custom Resistor-Transistor Logic (RTL) NOR gate with BC547 transistors to detect the optimal moisture level. Real-time soil conditions are indicated via Red (Dry), Blue (Wet), and Green (Optimum) LEDs. Verified using Proteus simulation.
